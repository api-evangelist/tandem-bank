# Tandem Bank

> Tandem Bank Limited is a UK app-only "greener" challenger bank (founded 2014, Blackpool) offering savings, cash ISAs, green home-improvement and home loans, mortgages, motor finance, and a credit card. As an FCA-authorised ASPSP it meets its PSD2 / UK Open Banking access-to-account obligations through a dedicated interface provided by Token (token.io), conformant to the OBIE Read/Write API Standard for Account and Transaction Information (AIS), Payment Initiation (PIS), and Confirmation of Funds (CBPII). Tandem is not a CMA9 bank and publishes no public Open Data reference API; production base URLs are issued per authorised TPP during Token onboarding and are not publicly published. The OpenAPI documents here are the shared OBIE standard specifications the interface conforms to, not Tandem-proprietary contracts.

## APIs
- [Account and Transaction Information API (AIS)](https://developer.token.io/): OBIE Read/Write account, balance, transaction, and product data. Scope: accounts.
- [Payment Initiation API (PIS)](https://developer.token.io/): OBIE Read/Write domestic/international/scheduled/standing/file payment initiation. Scope: payments. Requires x-idempotency-key.
- [Confirmation of Funds API (CBPII)](https://developer.token.io/): OBIE Read/Write confirmation of available funds. Scope: fundsconfirmations.
